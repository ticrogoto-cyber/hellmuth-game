Manifest (`src/data/sprites.ts`), klein, ohne Umlaute, Bindestrich statt
Leerzeichen. Fehlende Dateien sind unkritisch: die Entity behaelt ihre
gezeichnete Platzhalterform. Anker der Objekt-Sprites ist unten-mitte.
